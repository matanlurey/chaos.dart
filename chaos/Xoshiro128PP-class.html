Xoshiro128PP

Xoshiro128PP class final

A 32-bit Xoshiro128++ pseudorandom number generator (PRNG).

Xoshiro128++ is an all-purpose, high-quality PRNG that excels at generating both 32-bit random integers and floating-point numbers. It offers excellent statistical properties and a large state size (128 bits).

This is the recommended choice if you need a versatile generator for a wide range of applications, including simulations, games, and scientific computing.
